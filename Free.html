How to Install and Use Vae Victus Lite


Download ZIP

Step 1: Install Ollama

Ollama Allows You to Install the Language Models

  1. Visit the official Ollama website: https://ollama.com
  2. Download the installer for your operating system (Windows, macOS, or Linux).
  3. Run the installer and follow the on-screen instructions.
  4. Once installed, open a terminal or command prompt and verify the installation by running: ollama --version
  5. If the version number appears, Ollama is installed successfully.

Step 2: Install Vae Victus Lite

  1. Go to the Vae Victus Lite GitHub Repository:
    2. https://github.com/Rhill20/Vae-Victus-LLM-Manager
  2. Choose one of the following download options:
    • Option 1: Download ZIP Here Download ZIP
    • Option 2: Download ZIP From Github

      On the repository page, click on the Code button, then select Download ZIP.

    • Option 3: Clone via Git (For developers)

      If you have Git installed, you can clone the repository:

      git clone https://github.com/Rhill20/Vae-Victus-LLM-Manager.git
  3. Extract the ZIP File (if downloaded as a ZIP):
    • Locate the downloaded ZIP file in your **Downloads** folder.
    • Right-click the file and select **Extract All**.
    • Choose a location (e.g., **Desktop**) and click **Extract**.
  4. Create a Desktop Shortcut:
    • Navigate to the extracted folder.
    • Right-click **VaeVictusLite.exe** and select **Create Shortcut**.
    • Drag the shortcut to your desktop for easy access.
  5. Run Vae Victus Lite:
    • Double-click the shortcut on your desktop.
    • If you see a Windows Defender warning, click **More Info** ‚Üí **Run Anyway**.
